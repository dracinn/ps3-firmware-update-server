# PS3 Update Helper Homebrew

This is a small PS3 homebrew companion for PS3 Firmware Update Server. It checks the desktop app before you run the normal PS3 system update flow, then displays whether a compatible firmware is ready to install.

The helper does not flash firmware directly. Firmware installation still happens through:

```text
Settings > System Update > Update via Internet
```

## What It Checks

The homebrew reads these desktop app endpoints:

```text
http://<mac-ip>/api/status
http://<mac-ip>/api/firmware/manifest.json
```

The manifest is generated by the Kotlin app from the currently selected system profile, hardware status, main variant, and child variant. Firmware shown in `compatibleFirmware` has already passed the app's compatibility filter.

## GitHub Actions Build

This repository can build the helper with [FirebirdTA01/PS3DK](https://github.com/FirebirdTA01/PS3DK) in GitHub Actions.

Open **Actions > Build PS3 Helper > Run workflow**, then enter the Mac IP address that the PS3 should contact. The workflow uploads:

```text
ps3-update-helper.elf
ps3-update-helper.self
ps3-update-helper.fake.self
ps3-update-helper.pkg
ps3-update-helper.gnpdrm.pkg
SHA256SUMS.txt
```

The first run builds and caches PS3DK, so it can take a while. Later runs should reuse the cached toolchain.

## Local Build Requirements

- PSL1GHT, PS3DK, or another PS3 homebrew SDK with `ppu-gcc`
- Network access from the PS3 to the Mac running the desktop app

Build with a default server IP baked in:

```bash
make SERVER_IP=192.168.1.50
```

Clean build artifacts:

```bash
make clean
```

The Makefile produces `build/ps3-update-helper.elf`. Package it with your preferred PS3 PKG tooling/template for installation on the console.

When using PS3DK locally, you can also build with CMake:

```bash
source /path/to/PS3DK/scripts/env.sh
cmake -S . -B cmake-build \
  -G Ninja \
  -DCMAKE_TOOLCHAIN_FILE="$PS3_TOOLCHAIN_ROOT/cmake/ps3-ppu-toolchain.cmake" \
  -DSERVER_IP=192.168.1.50
cmake --build cmake-build
```

## Workflow

1. Open PS3 Firmware Update Server on the Mac.
2. Select the PS3 system profile and hardware status.
3. Select remote or local firmware and make sure a compatible `PS3UPDAT.PUP` is ready.
4. Start the server.
5. Run PS3 Update Helper on the PS3 and confirm the manifest says firmware is ready.
6. Set the PS3 DNS to the Mac IP shown in the desktop app.
7. Install through `Settings > System Update > Update via Internet`.

If the helper cannot reach the desktop app, check that the Mac and PS3 are on the same network and that macOS allows incoming connections on HTTP port 80.
